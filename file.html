Files

In this tutorial, we will learn about Python files, how to open , close files as well as how to write into files and read from files.

A file is a name given to the memory location on a computer that stores data, information.

To store, read, write, or work with files of an operating system is both necessary and easy with Python.

If we need to read from or write to a files, we can store files onto the local file system which is volatile and can be accessed every time using Python files.

In this article you will learn everything about file handling in Python which includes creating a file, opening a file, closing a file, writing and appending the files.

Opening a file

Python provides a open() function to open a files. This function returns a file object.

Syntax: open(file, mode)

The first argument file is a string containing the filename or an object representing a system path of the file to be opened.
The second argument is mode is an optional string that specifies the mode in which the file opened.

The default value for mode is 'r' which means open for reading in text mode, 'w' is for only writing. and 'a' opens the file for appending; any data written to the file is automatically added to the end. 'r+' opens the file for both reading and writing.

Files are opened in text mode by default, that means you read and write strings from and to the file. Python allows to open files in binary mode using ‘b’ character. In binary mode, data is read and written in the form of bytes objects. This is useful when files do not contain text.

Other common values for modes are:

Modes Meaning
‘r’ Open for reading in text mode
‘w’ Open for Writing in text mode
‘x’ Open for exclusive creation, failing if the file already exists.
‘a’ Open for writing, appending to the end of the file if it exists
‘b’ binary mode
‘t’ text mode (default)
‘+’ open a disk file for updating (reading and writing) 
# Open file in write mode.
f = open('test.txt', 'w')

# To Open a file in reading mode.
f = open('test.txt', 'r')

# Read and write file in binary mode.
f = open('test.txt', 'r + b')

Closing files

Once the file opened and done with all the operations. we must close files using the close() method. 

f = open('test.txt')
f.close()

It is good practice to use the with keyword when dealing with file objects. Using with statement, the files are properly closed after its operations, even if an exception is raised.

>>> with open('test.txt') as f:
        read_data = f.read()

>>> f.closed
True

Writing into a files

To write into a file, we need to open the file in a writing mode ‘w’ or in append mode ‘a’.

To create a new file or overwrite the existing file we can use mode ‘w’, while ‘a’ opens the file for appending; and any data written to the file is automatically added to the end.

To write the contents of string into a file, we can use write(string) method which writes and returns the number of characters written into a file. 

with open( 'test.txt', 'w') as f:
    f.write("This is first line\nThis is second line\n")

The above program will create a file name ‘test.txt’. if it does not exist, and if it exists, it is overwritten.

Reading from files

To read a file in a Python, we must open the file in reading mode.
To read a file’s contents, we can use read(size) method of files. Which reads some quantity of data and returns it as a string (in text mode) or in byte object (in binary mode).

The size is an optional numeric argument. if the size parameter is not specified, it reads and returns entire content. 

>>> f = open('test.txt', 'r')

>>> f.read(4)
'This'

>>> f.read(3)
' is'

>>> f.read()
'This is a the first line of the file\n'
'This is second line of the file\n'

>>> f.read()
' '

The read() methods returns empty string ' ' once the end of file is reached.

readline()

We can also use realine() method to read a single line from the file. This method reads a file until the newline character (\n). We can use readline when we want to read a file single line by line manner. 

>>> f.readline()
'This is a the first line of the file\n'

tell()

This method returns a current positi second line\n") on of a line in integer.

>>> f.tell() #returns a current position.
38

seek()

We can change the position of cursor/the file object’s position using seek() method. 


>>> f.seek(0) # bring file cursor to position 0.
0
