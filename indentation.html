Python Indentation.

In this article, we will learn importance of indentation in Pyhton, and how to use them.

A Python program is constructed from code blocks. A block of code is a piece of Python program statements or lines of code that are executed as a unit.

Python uses indentation to indicate a block of code. All statements of one block are intended at the same level indentation.

Python does not use the parentheses or curly braces for the "code blocks" like in other languages such as Java, C, C++ etc.

Indentation refers to the spaces at the beginning of a code line. The standard number of spaces for indentation in python is four spaces. Python will give you an error if you skip the indentation for example below. 

a = 5

# Error will occure in next line 
# because of single space indentation.

 print("The value of a is", a)

print("value of a is", a)

When we run this, we will get the following error. 

File "indent.py", line 3

    print("The value of a is", a)
    ^
IndentationError: unexpected indent

Did you Notice?, there is a single space at the beginning of the second line.

The error indicated by Python tells us that, the syntax of the program is invalid i.e. the program was not properly written.

What this means to you is that you cannot arbitrarily start new blocks of statements. Cases where you can use new blocks will be detailed in later chapters such as the control flow.
