Variables

In this tutorial, we will learn what is variable, how to create a variable in Python and what are the rules to follow while creating variables and then we will learn how to delete a variable.

One of the most powerful features of a programming language is the ability to manipulate variables. What is Variable? Variable is the name given to the memory location where data is stored. When a variable is created, the interpreter will create some space or memory to store the value of that variable. Variables are used to store data/values.

Everything in Python is Object. Every object in Python has an Indentity, a Type and a Value. Identity of the object is its address in memory and does not get change once it is created. Memory address or Identity of a variable can get using id(variable_name).
Type is nothing but the Datatype which means that, what type of data is store under the name of variable. It is a set of values and allowable operations on those values. Datatype of a variable can get using function type(variable_name).

How to create a Variable?

We know now, that a variable is a name refers to memory location and variables are also known as Indentifier and used to hold a value. Creating a variable in Python is very simple. Python's variables are dynamically typed, which means that we do not need to specify the type of variable. A variable does not need declaration or initialization, while creating or before adding value to it like in other languages such as C, C++, etc. There are some rules to follow while creating a variable in Python.

  • The name of the variable must start with a letter or with an underscore(_).
  • The name of the variable cannot start with a number.
  • The name of the variable cannot have a special character such as %, $, #, @, ! etc.
  • They can have only alphanumeric character and underscore (A to Z, a to z, 0-9, _).
  • Python is case sensitive language. for example, variable called " name " or " Name " are considered to be two different variables.

    • Variable Assignment

    Multiple Assignments

    You can assign multiple values to multiple variables in one line. but there must be the same number of arguments on the right and left side of the " = " operator.

    example.py
    
>>> a, b, c = 1, 2, 3
>>> print(a, b, c)
1 2 3

>>> a = 'String'

    Deleting Variable

    You can delete a variable using del statement. Which means The reference from the name to the value is deleted. Deleted variables can be reassigned to later as normal variable.

    example.py
    
>>> a = 10
>>> del a
>>> a
NameError: name 'a' is not defined.
