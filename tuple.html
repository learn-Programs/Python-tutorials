Tuple

In this article, we will learn what is tuple, how to create a tuple, how to access, change and delete a tuple elements, we will learn some popular tuple operations and methods as well.

We saw that list and string have many common properties, such as indexing and slicing operations. These two are an example of sequence data types (such as list, string, range).

There is another standard sequence data type constructor which is called tuple. Tuples are immutable sequences, typically used to store collection of heterogeneous data. Tuples are similar to the Lists. The only difference is that tuples are immutable sequence data types and Which means that elements of a tuple can not be changed once it has been created.

How to create a tuple?

A tuple is created by placing an elements inside parentheses (), separated by commas.

The parentheses are optional, it is the comma which makes a tuple, not the parentheses. But it is good programming practice to use them.

A tuple can have any number of elements which may have the same or different data types. Empty tuples are constructed by an empty pair of parentheses. 

>>> #Creating Tuple.

>>> #empty tuple
>>> my_tuple = ()

>>> #tuple using '()'
>>> my_tuple = (1,2,3)

>>> #tuple using a trailing comma.
>>> my_tuple = 1, 2, 3  #This is called tuple packing.

>>> #tuple with different data types.
>>> my_tuple = (1,'a',2.3)

>>> print(my_tuple)
(1,'a',2.3)

The tuple can be created using built-in tuple() or tuple(iterable) function. The iterable may be either a sequence such as string, list, or another tuple.

If iterable is a tuple, it is returned unchanged. 

>>> #creating a tuple using different methods.
>>> # tuple using built-in tuple() function.

>>> my_tuple = tuple((1,2,3))

>>> print(my_tuple)
(1, 2, 3)

>>> #iterable as string
>>> my_tuple = tuple('abc')

>>> print(my_tuple)
('a', 'b','c')

>>> #iterable as List
>>> my_tuple = tuple([1, 2, 3])

>>> print(my_tuple)
(1, 2, 3)

Creating a single value tuple

The tuple with one item is constructed by following a value with a comma, it is not sufficient to enclose a single value in parentheses.

Which means that, create a single value tuple you need to provide a value with a comma. if we create a single tuple with missing comma, then Python will consider that variable of string type and not a tuple since In tuple, comma who create a tuple and not parentheses for example as follows. 

>>> my_tuple = ("Hello")

>>> print(type(my_tuple))
< class 'str'>

>>> my_tuple = ("Hello",)

>>> print(type(my_tuple))
< class 'tuple'>

Sequence Unpacking

The sequence unpacking works for any sequence such as tuple, list, etc. Sequence unpacking requires the variables on the left side must be equal to the elements in the sequence on the right-hand side. 

#tuple packing
my_tuple = 1, 2, 3

#tuple unpacking
x, y, z = my_tuple

Access Tuple Elements

There are various ways to access the elements of the tuple.

1. Using Indexing

We can access elements of the tuple using indexing with index operator [] where index starts from 0.
The index of tuple with n elements will have index from 0 to n-1.

if you try to access elements outside the range of tuple index such as if tuple of length 5 with index from 0 to 4 and if you try to access tuple with index 5 or 6 then it will raise an IndexError.

The index must be an integer type; Datatypes other than integer will raise a TypeError. 

>>> my_tuple = ('a', 'b', 'c', 'd')

>>> #using index.
>>> print(my_tuple[1])
b

>>> print(my_tuple[5])
IndexError: tuple index out of range

>>> print(my_tuple[1.2])
TypeError: tuple indices must be integers or slices, not float.

2.Negative Indexing

Python allows negative indexing for its sequences. The index of -1 represents the last element, -2 to the second last element and so on. 

>>> # accessing elements of tuple using negative indexing.
>>> my_tuple = ('a', 'b', 'c', 'd')

>>> print(my_tuple[-1])
'd'

3.Using Slicing

We can access a range of elements in a tuple by using : in square brackets [] as tuple_name[start index: end index]. Just like any other sequences in Python, In tuple, the start is always included, and end index always excluded. 

>>> my_tuple = ('a', 'b', 'c','d')

>>> print(my_tuple[0:2])
('a', 'b')

>>> # using negative indexing
>>> print(my_tuple[-1: -3])
('d':'c')

Updating Tuple

Tuples are immutable sequence data type. This means that the elements of a tuple can not be changed once it has been created. Tuple can have elements with different data types such as list, dictionary, string, or even tuple as an element.

If the element is itself a mutable datatype like list or dictionary. then elements inside such datatype can be changed. 

>>> my_tuple = (1, 2, [3, 4])

>>> # changing elements inside list in tuple.
>>> my_tuple[2][1] = 10

>>> my_tuple
(1, 2, [3, 10])

Tuple Operations

We can use the + operator to combine two tuples. This is called concatenation. To repeat the elements in a tuple for a given number of times using * operator. 

#concatenation
tuple1 = (1, 2, 3)
tuple2 = (4, 5, 6)

print(tuple1 + tuple2)

output
(1, 2, 3, 4, 5, 6)

#Repeat
print(("Hello" * 2))

output
("Hello","Hello")

Tuple Membership Test

we can test if an element in the list is present in a tuple or not, using the keyword in and not in. 

>>> my_tuple = (1, 2, 3, 4)

>>> print( 1 in my_tuple)
True

>>> print( 5 in my_tuple)
False

>>> print( 5 not in my_tuple)
True

Iterating through a tuple

We can iterate through each element in a tuple using for loop. 

my_tuple = ('a', 'b', 'c')

for elem in my_tuple:

#output
# a
# b
# c

Deleting a Tuple

We can not delete or remove the elements in a tuple since tuples are immutable. But we can delete entire tuple using del keyword. 

>>> my_tuple = (1, 2, 3)

>>> del my_tuple
>>> my_tuple
NameError: name 'my_tuple' is not defined.
