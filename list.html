List.

In this tutorial, we will learn everything about list in Python, how to create a list, access list elements, update or delete list or elements in list, different list methods and built-in functions and list comprehension so, let get started.

Python has several datatypes, which used to group together other values. The list is one of the most frequently used and very versatile datatype used in Python. The List is sequence type which is finite ordered sets indexed by non-negative numbers.

How to create a List?

A list can be created by placing an item in a square bracket [ ], separated by commas. The list can be created by using the built-in function list(). A list can be a collection of elements with different types. which means a list can have different data types values.

example.py
#empty list.
list1 = []

#list with elements of same datatype --> 'int'.
list2 = [1, 2, 3]

#list with elements of different datatypes.
list3 = [1, 'a', 3.14]

Nested List.

The list can contain another list as an element, this is called a Nested List.

example.py

#nested list.
list_ = [3.14, 'Python', [2, 4, 6]]

Accessing elements from a list.

There are various ways to access the elements of a list.

1. Using Indexing

The elements of the list can be accessed by using indexing with the index operator [] where index start from 0. When the length of the list is n, index starts from 0 and goes to the length of n-1.
If you try to access an elements outside the range of list's index such as if a list of the length 5 then index will be from 0 to 4 and if you try to access list with index 5 or 6 then it will raise an IndexError.

The index must be an integer type; datatypes other than integer will raise an TypeError. The first element of a list is stored at the 0th index, second is at index 1, and so on.

So, the element at index i of list a can be accessed as a[i].

example.py

>>> #accessing elements in list 'a' using indexing.
>>> a = [11, 22, 33, 44]

>>> print(a[0])
11

2. Using Negative Indexing.

Python supports negative indexing for its sequences. The index of -1 represents the last element, -2 for the second last element, and so on. for example, let's access elements from above list 'a' using negative indexing.

example.py

>>> print(a)
[11, 22, 33, 44]

>>> print(a[-1])
44

>>> print(a[-3])
22

3. Using Slicing.

Sequences in Python also supports slicing. To access a range of elements in the list we can use slicing. The syntax for the same as follows.
list_name[start_index: end_index]
Here, the start is always included, and end index always excluded. Using slicing, let's access the range of elements of list 'b'.

example.py

>>> #accessing elements in list 'b' using slicing.
>>> b = [2, 4, 6, 8, 10, 12]

>>> b
[2, 4, 6, 8, 10, 12]

>>> print(b[2 : 5])
[6, 8, 10]

In the above slicing example, we print elements from the 2nd index to 4th index, but 5th will not be included.

Updating List.

List is a mutable sequence data type, which means that elements in a list can be changed. We can use the assignment operator (=) to change an element or range of elements in a list.

example.py

>>> #updating list.
>>> a = [1, 2, 3, 4]

>>> print(a)
[1, 2, 3, 4]  

>>> # change last element.
>>> a [3] = 5

>>> print(a)
[1, 2, 3, 5]

>>> a[0 : 2] = [11, 22]

>>> print(a)
[11, 22, 3, 5]

Adding elements in list.

List provides an append() method to add one element and extend() method to update a list with more than one elements.
We also can add an element at the given position i.e at given index using insert() method as list_name.insert(index, value).

example.py

>>> a = [1, 2, 3]

>>> #adding one element
>>> a.append(4)
>>> a
[1, 2, 3, 4]

>>> #adding several element
>>> a.extend([11, 21, 31])
>>> a
[1, 2, 3, 4, 11, 21, 31]

>>> #using insert method
>>> a.insert(1, 10)
>>> a
[1, 10, 2, 3, 4, 11, 21, 31]

List Operators.

To combine two lists, we can use + operator and this is called Concatenation.We can repeat a list for the given number of times using * operator.

example.py

>>> even = [2, 4, 6, 8]

>>> print(even + [10, 12])
[2, 4, 6, 8, 10, 12]

>>> print(["Hello"] * 3)
["Hello", "Hello", "Hello"]

List Membership Test

We can check if an element is present in a list or not, using in keyword.

example.py

>>> my_list = ['a', 'b', 'c']

>>> print('a' in  my_list)
True

>>> print('e' in  my_list)
False

Iterating through a list.

We can iterate through a list using for loop.

example.py

my_list = ['a', b', 'c']

for letter in my_list:
    print(letter)

Deleting list elements

Since lists are mutable sequence we can delete one or more elements from a list. There are various methods to delete elements in a list.

1. Using 'del' keyword.

del list_name[index]
here, the value at 'index' will be deleted from a list 'list_name'.

example.py
>>> my_list = [1, 2, 3, 4]
>>>
>>> #deleting one element.
>>> del my_list[1]
>>>
>>> my_list
[1, 3, 4]

2. Using pop() method.

The method pop() will remove the element/value at the given index position in the list and return it.
if no index is specified, it removes and returns the last item in the list.
list_name.pop(index)

example.py

>>> my_list = [11, 22, 33]

>>> my_list.pop(0)
11

>>> my_list
[22,33]

3. Using remove() method.

Remove the first occurrence of a value from the list using following syntax.

list_name.remove(value)
Let's try to remove the element from the above list, 'my_list'.

example.py

>>> my_list
[22, 33]
>>>
>>> my_list.remove(33) #here we provide a value from the list.
>>>
>>> my_list
[22]

4. Deleting List

We remove all elements from the list using clear() built-in function.
list_name.clear()

example.py

>>> my_list.clear()
>>> my_list
[]

We can delete the entire list using the del keyword.
del list_name

example.py

>>> del my_list
>>> my_list
NameError: name 'my_list' is not defined.

List Comprehension

List comprehension provides a concise way to create lists. List comprehension can be used when to make a list where each element is the result of some operations applied to each member of another sequence(such as list, tuple, range).

List comprehension consists of an expression followed by for statement inside square bracket.

example.py

#Create a list without list comprehension

squares = []
for x in range(10):
squares.append(x**2)

print(squares)

#output
[0, 1, 4, 9, 16, 25, 36, 49, 64, 81]

#Create a list using List comprehension.
squares = [x**2 for x in range(10)]

print(squares)

#output [0, 1, 4, 9, 16, 25, 36, 49, 64, 81]
example.py

# creating list of even numbers using list comprehension.
even = [x for x in range(1, 10) if x % 2 == 0]
print(even)
#Output[2, 4, 6, 8]
