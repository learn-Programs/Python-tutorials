Lambda-The Anonymous function

In this article, we will learn about Python's lambda functions, map, filter, and reduce function

Anonymous/Lambda function

In Python, an anonymous function is a function that is defined without a name.
Normal functions are defined using the def keyword, while anonymous functions are defined using the lambda keyword.
These functions are also called ‘lambda functions’.

Syntax: lambda parameters: expression

The parameters of the lambda are defined to the left side of the colon (:) while the expression on the right side.

The lambda functions can have any number of arguments but only one expression. The function evaluates the expression and return results. 

>>> # function definition with lambda function.
>>> # addition of two numbers.

>>> add = lambda a, b : a + b

>>> print(add(2, 3))
5

The unnamed above function object can be written in normal function as, 

#Normal function definition.

def add(a, b):
    return a + b

print(add(2, 3))

#Output
5

In Python, lambda functions are used in a higher-order function which takes a function as arguments. Lambda functions can be used along with built-in functions like map(), filter() and reduce() functions

Map function

The map() takes a function and a collection of items(list). It makes a new, empty collection, runs the function on each item in the original collection and insert each return value into the new collection.It returns the new collection type.

syntax: map(function, sequence) 

# Program to find square of numbers using a map() function.

nums = [1, 2, 3, 4]

squares = list(map(lambda x: x*x, nums))
print(squares)

#output
[1, 4, 9, 16]

Reduce function

The reduce() takes a function and a collection of items. it returns a value that is created by combining the items. It is used to apply a particular function passed in its argument to all of the list elements. To use reduce function in Python program we have import this fucntion from the functools module

Syntax: reduce(function, sequence) 

# importing reduce function.
import reduce from functools

Total = reduce(lambda a, x: a + x, [1, 2, 3, 4, 5])
print(Total)
    
#Output 
15

Filter function

The filter() function takes a function and a collection as a argument. It returns a collection of every item for which the function returned True.

Syntax: filter(function, sequence) 

#Program to find numbers greater than 3.

nums = [1, 2, 3, 4, 5]

my_list = list(filter(lambda x: x > 3, nums))
print(my_list)

#Output 
[4, 5]
