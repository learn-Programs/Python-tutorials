Dictionary

In this tutorial, we will learn about Python dictionary, how to create a dictionary, access, update, and delete elements from dictionary, we will also look at some useful dictionary operations and methods.

Another useful data type and currently the only one standard mapping type in Python is dictionary .
Python dictionary is an unordered collection of data and mapping mutable objects which means that we can change the values in dictionary.

How to Create a Dictionary?

Dictionaries can be created by placing a comma-separated list of key:value pairs with-in braces.

An item has a key and the corresponding value expressed as pair, key: value. The keys within dictionary must be unique, while the value can be of any datatype.

Key can be any immutable type such as string and numbers. Tuples can be used as keys if they contain only strings, numbers, or tuples. if a tuple contains any mutable object, it can not be used as keys. The list can not be used as key since lists are mutable datatype.
A pair of braces creates an empty dictionary. 

#empty dictionary.
my_dict = {}
    
#dictionary with integer keys.
my_dict = { 1: 'a', 2: 'b'}
    
#dictionary using dict() built-in function.
my_dict = dict({'name':'Radha', 'age': 7})
    
#using list of tuple of key and value as (key, value).
my_dict = dict([(1, 'apple'), (2, 'banana'), (3, 'orange')])

Accessing elements in dictionary

We know that indexing is used to access elements in other container types. But a dictionary is an unordered collection of values, it doesnâ€™t work with indexing, the dictionary uses keys to storing a value.
To access the values of dictionary, the key of that value can be used in square brackets.
Syntax: dictionary_name[key] 


#accessing elements in dictionary.
my_dict ={'name':'Radha', 'age': 7}
print(my_dict['name'])

#output
Radha

We can also use get() method which returns the value for a key.
Syntax: dictionary_name.get(key) 

print(my_dict.get('age'))

#output
7

The difference while using get() method is that it returns None instead of KeyError if the key is not found.

Udating elements in dictionary.

Dictionaries are mutable mapping objects, we can change or add new values in a dictionary. 


>>> my_dict = {1:'apple',2:'banana',3:'orange'}

>>> # changing value at key 1
>>> my_dict[1] = 'lemon'
    
>>> print(my_dict)
{1:'lemon', 2:'orange', 3:'banana'}
    
>>> # adding new item.
>>> my_dict[4] = 'papaya'
    
>>> my_dict
{1:'lemon', 2:'orange', 3:'banana', 4:'papaya'}

Delete or remove elements from dictionary

We can remove a particular value of a dictionary using pop() method as dict_var.pop(key). if key is in the dictionary, remove it and return its value. else return default and does not raise KeyError.


>>> #dictionary of programming languages.

>>> languages = {1: 'Python', 2: 'Java', 3: 'C', 4: 'C++', 5: 'R'}

>>> #value at key '1',i.e'Python' will remove and return.

>>> languages.pop(1)
'Python'

>>> print(languages)
{2: 'Java', 3: 'C', 4: 'C++', 5: 'R'}

The method, popitem() can be used to remove and return an arbitrary item(key, value) from the dictionary. 

>>> # returns in (key, value) 
>>> print(languages.popitem())
(5, 'R')

>>> languages
{2: 'Java', 3: 'C', 4: 'C++'}

del

we can also use del keyword as del dict_var[key]. it will remove a value from dictionary at the key. if key does not exist in dictionary, it raises keyError.

We can use clear() method to remove all items from the dictionary. we can also delete the entire dictionary using del keyword. for example, we can delete the value at key 3 from above languages dictionary as follows. 

>>> languages
{2: 'Java', 3: 'C', 4: 'C++'}

>>> #delete values using del.
>>> del languages[3]

>>> print(languages)
{2: 'Java', 4: 'C++'}

>>> #clear() method will remove all elements
>>> and return an empty dictionary.
>>> languages.clear()
>>> print(languages)
{}

>>> del languages
>>> print(languages)
    NameError: name 'languages' is not defined

Dictionary Operations

Membership Test

We can test if a particular key is present in a dictionary or not using in or not in keywords.

>>> squares = {1:1, 2:4, 3:9, 4:16}
>>> print(1 in squares)
True

>>> print(5 in squares)
True

Iterating Through a Dictionary

We can iterate through each key in dictionary using for loop.

# we can get the key and 
# values of dictionary using dict.items() method.

for key, values in squares.items():
print(key, values)

Output 

1 1
2 4
3 9
4 16

Dictionary Comprehension.

Python also supports dictionary comprehension.
Dictionary Comprehensions can be used to create dictionaries from key and value expression.

>>> # create squares dictionary with key as numbers and 
>>> # values as there squares using dictionary comprehension.

>>> squares = {x : x*x for x in range(2, 6)}
>>> squares
{2:4, 3:9, 4:16, 5:25}

Dictionary Methods

Most common dictionary methods are as follows. Methods can be accessed as dictionary_name.method(). excepts the iter(d) where 'd' is a dictionary.

Method Description
iter(d) Return an iterator over the keys of the dictionary.
clear() Remove all items from dictionary
copy() Return a copy of the dictionary
items() Return a list of dictionary items in (key, value) pairs
keys() Return a list of dictionary keys
values() Return a list of dictionary values
fromkeys(seq[, value]) Create a new dictionary with keys from sequence seq and values set to values.
