Datatypes

In this article, we will learn about different kind of data types in Python.

The data type is the type of variable. It may be an integer, string, or tuple.
Python variables do not need an explicit declaration of what type data, you stored in a variable. The declaration happens automatically when you assign a value to a variable.
There are two main types of Datatypes.

Immutable data types are those who can not change once you created them. While mutable data types are can be altered or change. You can check the data type of a variable using the built-in function type().

Numeric Datatype

There are three distinct numeric types: integers, floating-point numbers, and complex numbers in Python. Booleans are also a subtype of integers. 

>>> num =  10 #integer.

>>> print(type(num))
< class 'int' >

>>> float_num =  3.14 #floating point number.
>>> print(type(float_num))
< class 'float' >

>>> complex_num =  2 + 3j #complex number.	
>>> print(type(complex_num))
< class 'complex' >

String

Strings are an immutable sequence of Unicode code points. Strings are a set of characters represented in the quotation marks. In Python, we can use either pair of single or double or triple quotes to define a string. 

str_var = "Hello, World!"

Tuples are an immutable sequence of elements separated by commas and enclosed in parentheses (). Tuples can be sequence of same or combination of different data types.
A tuple is a read-only data structure as we don’t modify the size and value of the items of a tuple. 

#tuple
tuple1 = (1, 2, 3)

tuple2 = (1, 'a', 3.14)

List

A list contains items separated by commas and enclosed within square brackets []. Python list can have data of different data types. List are similar to arrays in C. 

#list
list1 = [2, 4, 6]

#List with Multiple Datatypes
list1 = [123, 'Python', 3.14]

Dictionary

Dictionary consists of key-value pairs. It is enclosed by curly braces {} and values can be assigned using square brackets []. 

dictionary = {'name': 'Advaita', 'age': 7}

Set

Sets are unordered collections of unique objects.
There two types of sets.
1. sets – They are mutable and new elements can be added once sets are defined. 

>>> basket = {'apple', 'banana', 'apple', 'orange', 'kiwi'}

>>> print(basket)
{'apple', 'banana', 'orange', 'kiwi'}

2. frozen set – They are immutable and new elements cannot be added after its defined. 

>>> b = frozenset('asdfagsa')

>>> print(b)
frozenset({'d', 'f', 'g', 's', 'a'})
