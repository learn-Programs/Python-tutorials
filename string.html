String

In this tutorial, we will learn about Python string from creating and accessing string, and how to change or delete strings, some popular string operation and methods. so, lets get started.

How to create String

A String is a sequence of characters that represent Unicode code points. In python, Strings can be created by enclosing the character or the sequence of characters in the quotes.
Python allows us to use single, double or triple quotes to the create a string.

example.py

>>> #creating string with 'my_str' variable.
>>> my_str = 'Python'

>>> #To check data type.
>>> print(type(my_str))
< class 'str' >

Strings may also be created from other objects using the str constructor.

example.py

>>> list_ = ['a', 'b', 'c']

>>> print(str(list_))
"abc"

Python doesn’t have a char type; instead every code point in the string object with length 1. The built-in function len() returns the length of a string.

example.py

>>> my_str = 'Python'
>>> len(my_str)
6

Accessing String

There are various ways to access the strings.

1. Using Indexing

Python supports indexing for its sequence type constructors. In Python, we can access the strings using indexing where the index starts from 0. If the length of the string is n; then the index of that string will start from 0 to n-1.

IndexError
When you try to access character which not in range of index, Python will raise an IndexError.

TypeError
Index is must be an integer type, but if you put an index of a different type than integer, then python will raise TypeError.

To access the single character in the string, we can use its index in square brackets [] for example as follows.

example.py

>>> my_str = 'Python'
>>> my_str[0]
P

2. Negative Indexing.

Python also allows negative indexing for its sequences. The index -1 represents the last element whereas -2 for second last and so on.

example.py

>>> my_str[-1]
n
>>> my_str[-2]
o

3. Using Slicing.

In addition to indexing, slicing is also supported by Python. While indexing is used to obtain individual characters, slicing allows you to obtain substring. In Python, the slice operator [] with a colon : is used to access the substring.
Syntax: Str_var[start Index : end index]
Here, the start is always included, and end index always excluded.

example.py

>>> my_str[1 : 4]
yth

>>> if start index is not specified the index defaults to '0'.
>>> my_str[: 4]
pyth

>>> To access entire string.
>>> my_str[:]
Python

How to Change or Delete a String

String is an immutable sequence data type which means we can not change the string once it has been created.
But, we can reassign new different Strings to the same string variable. for example.

example.py

>>> my_str = 'Python'

>>> my_str[0] = 'p'
...
TypeError:  'str'  object does not   support item assignment.

>>> my_str = 'Hello'

>>> my_str
Hello

We can not delete or remove characters from a string, but we can delete entire string using del keyword.

example.py

>>> del my_str
>>> my_str
...
NameError: name 'my_str' is not defined.

String Operations

Concatenation

The + operator is used to join two strings together. This is called concatenation.

example.py

>>> print('Hello' + 'World')
HelloWorld

Repetition

The * operator is used to repeat the string for given number of times.

example.py

>>> print('Hello'+* 3 )
HelloHelloHello

Iterating Thorugh String

We can iterate through the string using for loop.

example.py

my_str = 'Python'

#count the length of string using 'for loop'.
count = 0
for char in my_str:
    count = count + 1

print(count)
#Output: 6

String Membership Test

We can test if a substring exists in a string or not using the keyword in.

example.py

>>> print('e' in 'Hello' )
True

String Methods

Python supports various methods for string objects.
The following table contains some of the commonly used methods.
You can used methods as: str_var.method()

Method Description
str_var.capitalize() Return a copy of the string with its first character capitalized and rest lowercased.
str_var.lower() Returns lowercased string
str_var.upper() Returns uppercased string
str_var.find() Returns the index of first occurance of substring
str_var.index() Return index of substring
str_var.isalnum() Return True if character is alphanumberic
str_var.split() splits string from left
str_var.replace() Replaces Substring inside
str_var.join() Returns a concatenated string
str_var.format() formats string

String Formatting Method (str.format())

The str.format() method perform a string formatting operation.
The format string contains ‘replacement fields’ surrounded by curly braces {}.
Each replacement field contains either the numeric index of positional arguments or the name of a keyword argument.

example.py
    
>>> "The sum of 1 + 2 is {0}".format(1 + 2)
The sum of 1 + 2 is 3
