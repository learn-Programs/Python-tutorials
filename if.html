If..else Statement

Python supports various types of control statement.In this article, we will learn Python control statements such as if, if else, if elif, and nested if statement.

Control structures contain a group of other statements; they affect or control the execution of those other statements based on a decision.

Decision making is the most important characteristic of almost all the programming languages. Decision making allows us to run a particular block of code for a particular conditon and Python also supports the following control flow statements.

  • if Statement
  • if else Statement
  • if elif Statement
  • Nested Statements

if statement

The if Statement is used, when we want to execute a block of code only if a particular condition is satisfied.
Syntax 

if condition:
    block of code

if a condition is true then that block of code is executed. The condition of if statement can be any valid logical expression which can either be True or False. 

num = 4

if num % 2 == 0:
    print("Number is Even number.")

Output

Number is Even number.

if..else statement

The if-else statement provides else block of code.

if condition:
   if-block of code
else:
   else-block of code

The if statement evaluates the condition and when a condition is true, if-block of code is executed and when the condition is false, it executes the else-block of code.
The code inside the block should be indented, otherwise SyntaxError will raise. 

num = 5

if num % 2 == 0:
     print("Number is Even number.")
else:
     print("Number is odd number.")

Output

Number is odd number.

if..elif statement

The 'elif' is short for 'else if'. it allows to check multiple conditions. The elif statements can have a final else block, which is called if none of the if or elif expressions are True.

if condition:
   block of code
elif condition:
   block of code
.
.
else:
   block of code

Let's find the given number is multiple of 3 and odd number or not, using if elif statement. 

num = 21

if num % 2 == 0:
     print("Number is Even number.")

elif num % 3 == 0:
     print("The number is Odd Number and multiple of 3.")

else:
    print("The number is Odd number but not a multiple of 3.")

Output

The number is Odd Number and multiple of 3.

Nested if..else statement

We can have the if-else condition in other if-else statement. This is called nested if-else statement.

We can have any number of these statements inside one other. Indentation is the only way to figure out the level of nesting. 

if condition:
    if-block of code.

else:
    if condition:
       block of code.
    else:
        block of code.


# To find a given number is negative, zero or positive
# using nested if else statement

num = 2
if num >= 0:
    if num == 0:
        print("Number is Zero")     	
    else:
        print("Positive Number")
else:
     print("Negative Number")

Output

Positive Number
