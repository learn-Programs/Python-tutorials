Python Loop- While Loop

Loops are used in programming language to repeat a particular block code. In this tutorial, we will learn about while loop in Python.

What is while loop?

Loops are used to repeat a particular block of code. The while loop in Python is used to repeat the execution of a block of code as long as an expression (condition) is True.

Syntax:

while condition:
    block of code

This while loop repeatedly tests the expression and if it is true, executes the block of while loop. if condition or expression is false, the next section of code is executed. 

# Program to find the sum of 10 numbers using while loop.

n = 0
sum = 0

while n < 10:

    sum = sum + n
    n = n + 1

print("Sum = ",sum)

Output

sum = 45

In the above example of while loop. we print sum of first 10 numbers starting from 0. while loop has an increment operation where we are increasing the value of n by 1. This is a very important step.

The while loop must have an increment or decrement operation, else the loop will run infinitely.

Infinite while Loop

When the condition in while loop never becomes false, the while loop will never terminate and result in the infinite loop. 

#infinte loop.

n = 1

while n != 2:
    print(n)

Output

1
1
.
.
Infinte times

In the above example, while loop first evaluates the condition. Here, the condition is True, only when n is not equals to 2.
Since, the value of n is equal to 1, which is not equal to 2, which statisfies the condition.Now, program counter will enter into the while loop and excecutes the while block of code.

Here it will prints the value of n, which is 1. so program will prints the 1 and again check for the condition until the condition is false.
The program prints 1 indefinitely because inside the loop we are not updating the value of n. so the value of n will always remain 1 and condition will always remains True which leads to an infinite loop.

while loop with 'else'.

Python enables us to use the else with the while loop also. The else block is executed when the condition given in the while statement becomes false.

If while loop has break statement, then the else block will not be executed and the statement present after else block will be executed.

The following example is the combination of while loop with else statement, it will print a number as long as number is less than 3, otherwise it will print the else block of code.

# While loop with else.

n = 1
while n < 3:

    print(n)
    n = n + 1

else:
    print(f"{n} is not less than 3")

Output

1
2
3 is not less than 3
