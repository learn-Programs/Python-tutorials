Python Keywords.

In this article, we will learn about keywords in Python.

Python Keywords are special reserved words and cannot be used as ordinary identifiers or variable. Each keyword has a special meaning and a specific operation.

There are 35 keywords in Python 3.7. Following is the table of Python Keywords.

False class from or
None continue global pass
True def if raise
and del import return
as elif in try
assert else is while
async except lambda with
await finally nonlocal yield
break for not 

>>> # if you try to assign a value to keyword
>>> # then syntax error will occure. 
>>> for = 3
    
    for = 3
        ^
SyntaxError: invalid syntax

Notice that, when we try to assign a value to for keyword, the interpreter raised syntax error. so we can not use these keywords as variables or identifiers.
You can see this list any time by typing help('keywords') to the Python Interpreter. Reserved words are case sensative and must be used exactly as shown. They all are entirely lowercase except for False, None and True.
