Recursive Function

In this article, we will learn how recursive function works.

A recursive function is a function that calls itself in its definition.
For example, the following program finds factorial of a number using recursive function.

Syntax:

def func():
    . 
    func() # function call inside function itself.
    return 0
    
func() #function call outside of function body.


# recursive function.
def factorial(n):

    if n == 0:
        return 1
    else:
        return n * factorial(n-1)

print(factorial(5))

Output

120

Factorial of a number (n) is the product of all integers from 1 to n. for example factorial of number 5 is 1 * 2 * 3 * 4 * 5 = 120.
In the above example, the factorial() function call itself by decreasing the value of n number.
