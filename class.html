Classes

In this tutorial, we will learn Object-oriented Python programming and classes in Python.

Python is a multiparadigm programming language, such as Imperative where the main emphasis is on Statements, loops. Procedure Programming language where the main emphasis on functions.

Another very popular paradigm is an object-oriented programming(OOP) which emphasis on classes and object.

We can think of a class as a sketch of a house. it contains all the details about the floor, doors and windows etc. Based on these descriptions we build the house. Where the house is the object.

In object-oriented programming (OOP), the program is designed the data being operated upon rather upon the operation themselves. OOP allows us to decompose a problem into a number of entities called objects and then build's data and functions around those entities.

Object Oriented Programming.


What is OOPs?

Object oriented programming is an approach that provides a way of modularizing programs by creating a memory for both data and functions that can be used as templates for creating copies of such modules.

Objects

Objects are an entity or simply a collection of data that has variables and methods that act on those data. They may represent a person, a table of data, hardware devices like mouse, keyboard etc.

Everything in Python is an object, and almost everything has attributes and methods. In Python, object can be a variable, a data structure, a function or a method.

An Object is also called an instance of a class and the process of creating this object is called Instantiation.

Creating a Classes

A class is a way to bind data and its associated functions together. it allows data hidden from external use.

When we define a class, we are creating a new abstract data type that can be treated like any other built-in data type.

The class declaration describes the type and scope of its members. You should follow the below syntax to defined a class in Python program.

Syntax:

class ClassName:
     statements 1
     .
     .
     statements n

while the functions are defined using def keyword, the classes in Python are defined using keyword class followed by ClassName and a colon(:).

The first string is called docstring and has a brief description, about the class. The docstrings are optional but it is good practice to include a docstring.

When a class definition is entered, a new namespace is created and used as local scope.

example.py

class MyClass:

     """A simple class"""
     a = 10
     def f(self):
         print('Hello')

Creating an Object.

An object is the instantiation of a class. The object of the class can be created as follows.

Syntax:obj = ClassName()

Here, the new instance object named ‘obj’ created. which allows to access attributes of objects.

The Attributes can be variables or methods in class. Functions defined in a class are called methods.

example.py

# Creating Object.
x = MyClass()

print(x.f())
# Output
Hello

print(MyClass.a)
#Output 
10

The MyClass.a and MyClass.f are attributes of the class and x.f is a method object.

We can delete the object using the del keyword.

example.py
   
>>> del x
>>> x
NameError: name 'x' is not defined.

Method

The method is a function that is associated with an Object. In Python, a method is not unique to class instances.

Constructors

A constructor is a special member function for automatic initialization of an object. Whenever an object is created, the special member function i.e the constructor will be executed automatically.

A constructor function is different from all other member functions in a class because it is used to initialize the variables. It is called constructor because it constructs the values of data members of the class.

In Python, a class function that begins and ends with a double underscore (__) are called special function.

The __init__() function is special function, is constructor in Python. This method is the most important method in a class. This is called when an instance (object) of a class is created.

We can pass any number of arguments at the time of creating the class object, depending on the __init__ definition.

example.py

>>> class MyClass:
        def __init__(self):
            print("It will print when object is created.")

>>> #creating object      
>>> x = MyClass()

 It will print when object is created.

The 'self' Parameter

All methods in class must have self as their first parameter, which is referring to the instance. but you need not give any value for this parameter when you call the method. This is because, whenever an object calls its method, the object itself is passed as the first argument.

In above example, the call x.f() is exactly same as MyClass.f(x).

example.py

class ComplexNumber:

    def __init__(self, real, img):
        self.real = real
        self.img = img

    def complexNum(self):
        print("{0} + {1}j".format(self.real, self.img))

# Creating Object of ComplexNumber.
C = ComplexNumber(2,3)
C.complexNum()

Output

2 + 3j

Built-in attributes

Instances of a class have attributes, which is data associated with them.
Every object has attributes. These can be accessed by putting a dot operator (.) and attribute name after an instance.

example.py

>>> C.real
2

Python has the following in-built functions for class attributes.

Functions Description
getattr(obj, name, default) It is used to access the attribute of the object
setattr(obj, name, value) It is used to set an value to the specific attribute of an object.
delattr(obj, name) it is used to delete a attribute.
hasattr(obj, name) it is used to check if an object has a specific attribute or not.
example.py

class Student:

def __init__(self, name, age):
    self.name = name
    self.age = age

#creating object of class student.
s = Student('Harry', 12)

print(getattr(s, 'name')

print('before setting age', s.age)

setattr(s, 'age', 14)
print('after setting age', s.age)

print(hasattr(s, 'id'))

print(delattr(s, 'age'))
print(s.age)

Output

Harry

beforesetting age is 12
after setting age is 14

False

None

print(s.age)

Class Methods

Methods of objects are called an instance of a class, which takes a parameter 'self', which is points to an instance of a class when method is called.

The Class methods are called by a class, which is passed to cls parameter of the method. The class methods work the same way as methods in a class, except that when class method is called it passes the class as the first argument.

example.py

Class A:

    multiplier = 2
    @classmethod
    def f(cls, x):
         return cls.multiplier * x

#Object of class A
a = A()
print(a.f(3))

Static Method

Static methods of a Python class is similar to the normal method in python which does not take a parameter like self. It is just like any other function defined in the class but it is callable without creating the instance of the class. The method belongs to the class but doesn’t use the object itself at all.

They use static method decorator.

example.py

Class A:

    x = 10
    @staticmethod
    def f(name):
        print('Hello', name)
a = A()
a.f('World!')

Output

    Hello World!
