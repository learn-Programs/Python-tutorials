Python First Python Program

In this article, we will write your first Python program.

Now, that we have Python downloaded and we have a basic introduction of Python, we can write our first Python program. Let’s create a very simple program called “Hello World”.
Type the following code in an IDE and save it as helloWorld.py

example.py

# Your first python program.
print("Hello, World!")

Then, run the file, You will get the following output.

Hello, World!

Congratulations! You just wrote your first python program.
