Functions

Functions are the most important features of the Python programming language. In this article, we will learn how to create a function, use function in python programs with return and pass statement.

Functions are the most important features of the Python programming language. A function is a block of reusable code which is used to perform a specific task.

The function helps break our program into a smaller piece of code that is executed as a unit. Functions make our program more organized and manageable and it avoids repetition and makes code reusable.

Python has many built-in functions like print(), len() which we use until now. But you can also create your own functions in python. These functions are called User Defined Function. In order to create a user define function we need to first define a function. So, let's see how we can do that.

Function Definition

You can define a function using following syntax.

Syntax 

def function_name(parameters):
    """Docstrings"""
    statements
    return [expression]

The keyword def is used to define a function and it is called a "function definition".
It must be followed by the "function name" and the "parentheses" () where you can define a list of "parameters", the parameters are optional, and colon(:) that ends the line.

The statements that form the body of the function start at the next line and must be indented.

The first statement of the function body can be a string literal. This is called the functionâ€™s "documentation string" or docstring. The docstring is optional but it is good practice to include docstrings in code. 

# defining simple function.

def hello():
    """This is simple function."""
    print("Hello, world!")

Calling Function

Once a function is defined, we can call the functions from another function or python prompt. The function definition does not execute the function body, this gets executed only when the function is called.

To call the function, use the function name followed by the parentheses with appropriate parameters.

Following is the example of calling above function hello() that we just create. 

hello() # function call.

Output 

Hello, World!

Docstrings

In the above example, we have a docstring immediately below the function header.
This docstring is must be the first line and should always be a short, concise summary of the function.

This line should begin with a capital letter and end with a period. We can get this docstrings using __doc__ attribute of the function. 

>>> hello.__doc__
This is simple function.

Function with Parameters

A function can take a various number of parameters. These parameters are just like variables except that the values of these variables are defined when we call the function.

Parameters are specified within the pair of parentheses in the function definition, separated by commas.

Parameters are named that appear in the function definition, where as values that passed to a function when calling it, are called arguments. Parameters define what type of arguments a function can accept.

The following function takes a string as input parameter.

# defining function with parameter.

def hello(name):
    """This is function with string parameter."""
    print("Hello", name)

name = John
hello(name)# calling function

Output 

Hello John

Functions with Default argument values

Python allows to initialize the arguments at the function definition using the assignment operator (=). This is called default argument values.

The most useful form is to specify a default value for one or more arguments. If the value of argument is not provided then this default value is used.

# function definition.

def greet(name, greeting = 'Welcome'):
    """This function greets"""

    print(greeting, name)

# function call.
greet('Radha', 'Good Moring')
greet('Radha')

Output 

Good Morning Radha
Welcome Radha

Keyword Arguments

Functions can also be called using Keyword Arguments of the form kwarg = value.
In Python, we can use the name (keyword) instead of the position (which we have been used) to specify the arguments to the function. 

def func(a,b = 1):
    print(a + b)

# function call with positional arguments.
func(1)
func(3, 4)

# function call using keyword arguments.
func(a = 2, b = 3)

Output 

2
7
5

Arbitrary Argument lists

Python allows to define a function that can take any number of parameters. This function can be called with an arbitrary number of arguments. These arguments are collected as a tuple.
This can be achieved by using stars.

def func(*names):
    
    print(type(names))
    for name in names:
        print(name)

func( 'a',  'b', 'c', 'd')

Output

 <class 'tuple'>
a
b
c
d

When we declare a starred parameter as *args, then all the positional arguments are collected as a tuple.
when we declare a parameter with two-star as **kwarg, then all the keyword arguments are collected as a dictionary.

The return Statement

The return statement is used to return from a function call.

Syntax: 

return [expression_list]

The return statement leaves the function with evaluated expression value. If there is no expression in the statement or the return statement is not present in the function, then the function will return the None Object. 

def isEven(num):
    
    if num % 2 == 0:
        return True
    else:
        return False

 # Function call.
isEven(4)
isEven(5)

Output 

True
False

Pass Statement

The pass statement does nothing. it is used as place-holder for a function or conditional body when you create a function with no statements. They cannot have an empty body since interpreter will raise an error, so we can use the pass statement to construct a body that does nothing.

Syntax: 

pass


def function():
    pass

This can be used for creating minimal classes. 

class MyEmptyClass:
     pass
